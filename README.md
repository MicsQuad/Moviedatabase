
# Movie Maze

## 1. Technical Solutions

### Frontend:
* React
* Vite
* CSS
* JavaScript
* HTML
* Axios

### Backend:
* Node.js
* Express
* MongoDB Atlas
* JWT

### Deployment
* Render

## 2. Clone the repo

```bash
git clone https://github.com/MicsQuad/MovieDatabase.git
cd MovieDatabase
```

## 3. How to Run

1. Install dependencies on root folder:

```bash
npm  install
```

2. On  the  client  folder and on the server folder run:

```bash
cd client
npm install

cd ../server
npm install
```
 
3.  To  run  it  all:

```bash
cd ..
npm run dev
```
